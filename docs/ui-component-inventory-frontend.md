# UI Component Inventory - Frontend

## Scope
- Page components: ~19 JSX files under `src/pages`
- Reusable components: ~60 JSX files under `src/components`

## Major Categories
- Layout/navigation: dashboard layout, protected app shell
- Operational screens: inbound, stock, issue, receive, dispatch, opening stock, reports, boiler, settings, send-documents
- Domain components:
  - `components/inbound`, `components/issue`, `components/receive`, `components/stock`, `components/dispatch`, `components/reports`
- Shared foundation:
  - `components/common/*` (inputs, filters, toolbars, permission gate)
  - `components/ui/*` (button, input, select, dialog, table, badge, card)

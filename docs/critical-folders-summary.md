# Critical Folders Summary

- `apps/frontend/src`: UI features, app shell, contexts, API clients
- `apps/backend/src/routes`: authoritative business and integration API surface
- `apps/backend/prisma`: persistent schema and migration history
- `apps/local-print-service`: local OS-level print bridge
- `apps/print-client/src-tauri`: desktop runtime and packaging config
- `docker/` + root compose files: infra bootstrap and environment composition
